Biometria (integração com catraca/leitor externo)

Duas formas de preencher o ID biométrico: (1) clique em "Capturar pela catraca" e peça pro aluno tocar o dedo no leitor da catraca — o ID lido preenche o campo abaixo automaticamente; ou (2) se você tiver um leitor de digital separado conectado neste computador, cole aqui o ID que o software dele mostrar. Em qualquer um dos dois casos, revise o valor e clique em "Salvar ID biométrico" pra confirmar.

Acesso pelo totem (QR pessoal / reconhecimento facial)

Gere o link/QR pessoal do aluno para ele usar no totem ao lado da catraca (celular). O reconhecimento facial pode ser cadastrado pelo próprio aluno no totem, ou por aqui mesmo usando a câmera deste computador (útil para cadastrar na hora da matrícula, sem levar o aluno até o totem).

Configurações

Identidade do app

O nome aparece na tela de login e na barra lateral. "Licenciado para" some abaixo do nome, em letra menor — deixe em branco pra não mostrar nada.

Treino em app externo

Usado na aba "Treino" do perfil quando o aluno estiver marcado como "treino em outro aplicativo".

Portal remoto do aluno

Usado no botão "Agendar avaliação" do portal remoto — abre uma conversa de WhatsApp já com uma mensagem pronta. Deixe em branco pra ocultar esse botão.

Link do portal do aluno

Compartilhe este link com os alunos (site, bio do Instagram, WhatsApp) — de lá dá pra pagar contas em atraso, trocar de plano, ver o treino e se cadastrar, sem precisar vir à academia. Nunca libera a catraca.

Ordem dos menus

Use as setas para mudar a ordem dos itens na barra lateral. A ordem salva aqui vale para todos os usuários do sistema.

    Backup

    O sistema já gera um backup automático a cada 24h guardado no próprio servidor, mas isso não é uma cópia de segurança de verdade (some se o servidor reiniciar/redeployar). Use o botão abaixo pra baixar um backup completo agora — em JSON, direto pro seu computador — e guarde num lugar seguro (Google Drive, e-mail, etc.) de tempos em tempos.
